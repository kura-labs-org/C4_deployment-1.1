## Deployment Instructions:
1. Create a diagram of the pipeline 
2. Log onto the instructor's Jenkins server and run the build for the application
3. Observer the pipeline stages and document what occurred
4. **IF** the pipeline is successful, download the application files from your repository and proceed to the next step: https://scribehow.com/shared/How_to_Create_and_Deploy_a_Python_URL_Shortener_on_AWS_Elastic_Beanstalk__MS9pB8lfRaGFiKAq2FU-cw
5. **IF** the pipeline fails, figure out the issue and write a report of the issue in your documentation
6. Fix the issue and then run your pipeline again
7. Then proceed to the next step: https://scribehow.com/shared/How_to_Create_and_Deploy_a_Python_URL_Shortener_on_AWS_Elastic_Beanstalk__MS9pB8lfRaGFiKAq2FU-cw
